# TYPO3 extension ``kesearch_categories``

## Introduction
Extension provides categories for ``EXT:ke_search``-filteroptions.
Categories are used as ``EXT:ke_search`` - indexable tags for pages and news.

## TYPO3 compatibility

* TYPO3 6.2 LTS
* TYPO3 7.4
